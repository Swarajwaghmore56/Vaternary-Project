This project is a Veterinary Management System developed during my internship. It helps veterinary clinics manage their daily operations, including patient records, appointments, and treatments. The system is built using JavaScript, PHP, MySQL, and XAMPP for local development.

Technologies Used
Frontend: JavaScript, HTML, CSS
Backend: PHP
Database: MySQL
Server Environment: XAMPP

Features
Add, view, update, and delete patient records
Manage appointments and treatment history
Search and filter records for better data management

Project Structure : 
      Vaternary/
│
├── index.php          # Home page
├── patients.php       # Manage patient records
├── appointments.php   # Appointment management
├── assets/
│   ├── css/           # Stylesheets
│   └── js/            # JavaScript files
├── database.sql       # Database backup
└── README.md          # Project description
